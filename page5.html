2Fasti

Validate and confirm the change and deletion of legally and financially significant data in SQL BD using blocking technology.

- Voting results,
- Sociological surveys,
- State registers,
- Accounting databases,
- Hotline Messages

All these are databases containing records that can not be changed or deleted.

However, such manipulations with the data still sometimes occur and entail legal and financial consequences.

The product is a software - an agent working alongside a database that needs to be verified.
Agent receives data from database triggers about changes and uses a smart contract to make the corresponding entry in a public or private blockchain.
  • The agent does not have access to the data and does not need a user with any rights.
  • All the preparatory actions for starting the control are simple and are carried out by the usual means of the sql administrator with root rights.
  • In the event of any recorded incidents with data manipulation, the database administrator with root rights (and only him) is fully responsible for such manipulations.
