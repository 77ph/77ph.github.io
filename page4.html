Licenses Control

Using ethereum smart contracts to setup and control licenses for the network devices

License management system for networks of devices with variable parameters.

Devices can be elements of the urban infrastructure (parking lots, ATMs, traffic light controllers, etc.)

Devices must meet the following requirements:
- A small amount (<100,000)
- time for changing parameters is not critical ~ 15 minutes

Device management software - linux server. 

Each device contains a contract with a unique address from a blockchain. Only the owner of the contract can manage all devices.

prototype - device for monitoring the quality of video streams in the operator's network
licensed parameters - number of threads, time of use
